# ComfyUI ExpoBooth Prompt

A lightweight ComfyUI custom node for generating structured exhibition booth prompts.

This public repository is intended for installation and deployment compatibility with ComfyUI environments, including hosted platforms such as RunningHub.

## Purpose

`ComfyUI ExpoBooth Prompt` provides a simple prompt-building node for exhibition booth concept generation workflows.

It helps convert selected booth parameters into a structured text prompt that can be connected to image-generation, LLM, or automation workflows inside ComfyUI.

This public version contains only the deployment-safe node files required for ComfyUI compatibility.

## Repository status

This is a public deployment version.

Internal ExpoBooth.ai production workflows, private prompt systems, advanced automation logic, client-specific methods, and commercial workflow structures are not included.

## Features

- ComfyUI custom node for exhibition booth prompt generation
- Structured booth prompt output
- Parameter-based booth concept direction
- Optional frontend styling through the `web` folder
- Designed for use in AI concept-generation workflows
- Compatible with local ComfyUI installations and hosted ComfyUI environments
